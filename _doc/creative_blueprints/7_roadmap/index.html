Post-MVP Roadmap

What Comes Next

3 Capability Pillars
Pillar 1 — Engagement

Multiplayer
Live Battles

Real-time head-to-head word battles against friends or matched opponents on a shared board. Quick-match queue, challenge-a-friend invites, and a dedicated Battle Wins leaderboard.

  • 3-minute live rounds, same board seed
  • Buzz Bot fallback if no match in 30s
  • Preset bee emotes — family-safe
  • Honey Coins + XP awarded post-battle
Pillar 2 — Retention

Seasonal Events
& Special Worlds

Limited-time holiday themes that transform the world map and board. Dedicated event quest lines, exclusive earn-only cosmetics, and a reusable event framework so new events are configuration, not code.

  • Dynamic CSS skin system per event
  • 5–7 quest line with Grand Prize chest
  • Event-only Buzz outfits, tiles, boards
  • Two launch templates: Summer Hive Fest & Winter Frost Hive
Pillar 3 — Virality

Social & Friends
System

Friend lists, a friends-only leaderboard, daily Honey Coin gifts, and challenge invites with shareable deep links. Turns individual retention into network retention — every active friend is a re-engagement trigger.

  • Add friends by username or invite link
  • 1 gift (25 coins) per friend per day
  • Challenge a friend to beat your score
  • Unified notification inbox for all social activity

Design principle: Every post-MVP pillar is additive — it deepens the core loop without changing it. Multiplayer adds urgency. Seasonal events add novelty. Social adds network gravity. Together they compound Day-30 retention without paywalling the experience.

All rewards earn-only. No paywalls.
